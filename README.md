# IMA-5-Factor-Model
Our take on the Fama-French five factor model. Developed to assist the portfolio managers of the Illinois Investment Academy identify targets for outperformance. Currently uses data from Tuck School of Business, hopefully we can transition to WRDS soon.

## Sources:
Five-factor Model: https://www.sciencedirect.com/science/article/pii/S0304405X14002323  
Fama-French Paper on additional factors to the CAPM model: https://www.sciencedirect.com/science/article/pii/0304405X93900235  
Data Source for now (up until 8/29/2025): https://mba.tuck.dartmouth.edu/pages/faculty/ken.french/data_library.html#Research
